#  App de Lista de Tarefas com Tema Dinâmico

Projeto desenvolvido em **React** utilizando **Styled Components**, permitindo gerenciar tarefas de forma simples, moderna e responsiva.

##  Funcionalidades

*  Adicionar novas tarefas
*  Marcar tarefas como concluídas
*  Remover tarefas
*  Filtrar tarefas:

  * Todas
  * Ativas
  * Concluídas
*  Alternar entre tema claro e escuro
*  Interface responsiva
*  Estilização com Styled Components

---

##  Tecnologias Utilizadas

* React

##  Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/lista-tarefas.git
```

### 2. Acesse a pasta do projeto

```bash
cd lista-tarefas
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Instale o Styled Components

```bash
npm install styled-components
```

### 5. Execute o projeto

```bash
npm run dev
```

##  Como Utilizar

### Adicionar uma tarefa

Digite a descrição da tarefa no campo de texto e clique em **Adicionar**.

### Concluir uma tarefa

Clique na caixa de seleção ao lado da tarefa para marcar como concluída.

### Remover uma tarefa

Clique no botão **Remover** correspondente à tarefa.

### Filtrar tarefas

Utilize os filtros disponíveis para visualizar:

* Todas as tarefas
* Apenas tarefas ativas
* Apenas tarefas concluídas

### Alternar tema

Clique no botão localizado no cabeçalho para alternar entre:

*  Tema Claro
*  Tema Escuro
